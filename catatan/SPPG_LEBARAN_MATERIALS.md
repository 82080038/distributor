## üè∑Ô∏è KODING SISTEM

### Format Kode Lebaran:
```
SPPG-L[Kategori][Subkategori][Nomor]
Contoh: SPPG-LK01-001 (Kue - Kue Nastar 1)
```

### Kategori Lebaran:
- **LK** = Kue Lebaran (Cakes & Cookies)
- **LB** = Bahan Kue (Baking Ingredients)
- **LM** = Minuman (Beverages)
- **LT** = Ketupat & Lauk (Main Dishes)
- **LBM** = Bumbu Khas (Special Spices)
- **LS** = Snack & Jajanan (Snacks & Light Meals)

## üì± IMPLEMENTATION GUIDE

### Database Structure:
```sql
create table sppg_lebaran_materials (
    id bigint unsigned auto_increment primary key,
    material_code varchar(50) unique not null,
    material_name varchar(255) not null,
    category enum('LK', 'LB', 'LM', 'LT', 'LBM', 'LS') not null,
    subcategory varchar(50) not null,
    product_type varchar(100) null,
    package_size varchar(50) not null,
    unit varchar(20) not null,
    shelf_life_days int not null,
    storage_requirements json null,
    halal_certification varchar(100) null,
    bpom_number varchar(100) null,
    supplier_id bigint unsigned null,
    is_seasonal boolean default false,
    peak_season_start varchar(20) null,
    peak_season_end varchar(20) null,
    is_active boolean default true,
    created_at timestamp default current_timestamp,
    updated_at timestamp default current_timestamp on update current_timestamp
);
```
