## üåæ STANDAR NUTRISI BGN

### Target per Porsi (Rp 10.000):
- **Kalori**: 2000-2500 kcal
- **Protein**: 15-20% dari total energi
- **Lemak**: 25-30% dari total energi
- **Karbohidrat**: 45-55% dari total energi
- **Serat**: 25-30 gram per hari
- **Vitamin & Mineral**: Sesuai AKG

## üìä KALKULATOR MENU SPPG

### Formula Perhitungan Gizi:
```sql
CREATE VIEW v_sppg_menu_nutrition AS
SELECT 
    material_code,
    material_name,
    category,
    subcategory,
    -- Data nutrisi per 100g
    protein_per_100g,
    carb_per_100g,
    fat_per_100g,
    fiber_per_100g,
    calories_per_100g,
    -- Hitung kebutuhan per orang per hari (target dewasa)
    CASE 
        WHEN target_group = 'dewasa' THEN 
            (protein_per_100g * 50 / 100) * 7 + -- 50g protein/hari
            (carb_per_100g * 300 / 100) * 7 + -- 300g carb/hari
            (fat_per_100g * 65 / 100) * 7 + -- 65g lemak/hari
            (fiber_per_100g * 30 / 100) * 7 -- 30g serat/hari
        WHEN target_group = 'anak' THEN 
            (protein_per_100g * 30 / 100) * 7 + -- 30g protein/hari
            (carb_per_100g * 130 / 100) * 7 + -- 130g carb/hari
            (fat_per_100g * 25 / 100) * 7 + -- 25g lemak/hari
            (fiber_per_100g * 25 / 100) * 7 -- 25g serat/hari
    END as daily_protein_grams,
    calories_per_100g * 2000 / 100 * 7 as daily_calories,
    -- Hitung kebutuhan per minggu
    (protein_per_100g * 50 / 100) * 7 * 7 as weekly_protein_grams,
    (carb_per_100g * 300 / 100) * 7 * 7 as weekly_carb_grams,
    (fat_per_100g * 65 / 100) * 7 * 7 as weekly_fat_grams,
    (fiber_per_100g * 30 / 100) * 7 * 7 as weekly_fiber_grams
FROM sppg_materials
WHERE is_active = TRUE;
```
